# postman-api-tests
Projeto desenvolvido durante meus estudos em QA, aprofundando conhecimentos
em testes de API, métodos HTTP, status codes e automação de validações.

## Sobre o projeto

Este projeto demonstra a criação de testes funcionais para uma API REST,
cobrindo operações do CRUD Create, Read, Update, Delete e validações
de status.

## Ferramentas utilizadas

- **Postman** — criação e execução das requisições e testes
- **JavaScript** — escrita dos scripts de teste
- **API de treino:** JSONPlaceholder (https://jsonplaceholder.typicode.com)

## Tipos de validação

- Validação de **status code** (200, 201)
- Validação de **conteúdo** da resposta (campos e valores)
- Validação de **tempo de resposta** (performance)

## Como usar

1. Importe o arquivo `.json` no Postman (botão "Import")
2. A collection "Postman API Tests" será carregada
3. Execute as requisições e veja os resultados na aba "Test Results"
